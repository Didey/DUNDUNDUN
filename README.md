Platformer made in Java, just for fun. Maybe something more in the future.

developed by TheDrink with help from Dx724

#Screenshots

![Signs!](http://i.imgur.com/TScjPCW.png)


#Controls
Pause - e

Debug - F5

Move Left - a

Move Right - d

Jump - Space (or w)

Console - /

Quit - Escape
